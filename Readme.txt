Fisierul .mdf se gaseste in folderul Baza de date.

Aplicatia a fost optimizata pentru a lucra cu calendarul Windows setat pe formatul datei in Romania. 
In cazul in care se foloseste alt tip de format se schimba dimensiunea calendarului afisat in aplicatie si se poate suprapune cu alte elemente.

Sunt 3 conturi create in aplicatie:
1) Cont: Ghinea , parola : Razvan
2) Cont: Iordache, parola : George
3) Cont: Scurtu, parola : Petru
